Crosswalk Manifest Demo 2

Download app named like "manifestdemo2.apk" from https://github.com/crosswalk-project/demo-express/releases (It might be included in .zip files in this page), then install it.
This demo shows following features can be configured by manifest.json:
  • Icon: A gif format icon (green background color)
  • Xwalk_version: 1.0.1
  • Orientation: Portrait mode
  • Splash Screen: Green background color
  • Splash Screen: Ready when the DOM and CSSOM have been constructed
  • Splash Screen: Portrait mode
  • Display: The application displays in the same manner as a native application. The standard UI elements from the context (e.g. the device's status bar) remain visible

Expected results

  1. Install app successfully, no error, no warning message
  2. Display text "Manifest GIF" with green background normally and the version is "1.0.1"
  3. Launch the app successfully and display "Splash Screen Portrait" with green background in Splash Screen
  4. Display some descriptions about manifest configuration with portrait screen
  5. Display status bar

This sample demonstrates manifest members "icons", "xwalk_version" "orientation", "display", "xwalk_launch_screen"

  • Test values include: gif format icons, three digits version, portrait orientation, standalone display, user-interactive ready_when and portrait
  • Standalone: In this mode, the application displays in the same manner as a native application. The standard UI elements from the context (e.g. the device's status bar) remain visible
  • Xwalk_launch_screen members
    • Ready_when: The application readiness state at which to hide the launch screen
      • User-interactive:
        • The first visually non-empty paint has occurred
        • The DOM and CSSOM have been constructed
    • Portrait: The launch screen to use for portrait mode
