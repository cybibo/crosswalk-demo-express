Crosswalk Manifest Demo 2

This demo shows following features can be configured by manifest.json:
  • Icon: A gif format icon (green background color)
  • Xwalk_version: 1.0.1
  • Orientation: Portrait mode
  • Splash Screen: Green background color
  • Splash Screen: Ready when the DOM and CSSOM have been constructed
  • Splash Screen: Portrait mode
  • Display: The application displays in the same manner as a native application. The standard UI elements from the context (e.g. the device's status bar) remain visible
