Crosswalk Manifest Demo 1

This demo shows following features can be configured by manifest.json:
  • Icon: A png format icon (blue background color)
  • Xwalk_version: 1.0
  • Orientation: Default mode
  • Splash Screen: Blue background color
  • Splash Screen: Ready when all the resources have been loaded
  • Splash Screen: Can switch screen between portrait and landscape modes
  • Display: The application is displayed so that it fills the entire area of the screen, with none of the runtime's UI elements ("chrome") visible
