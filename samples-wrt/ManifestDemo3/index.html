Crosswalk Manifest Demo 3

Download app named like "manifestdemo3.apk" from https://github.com/crosswalk-project/demo-express/releases (It might be included in .zip files in this page), then install it.
This demo shows following features can be configured by manifest.json:
  • Icon: A bmp format icon (turquoise background color)
  • Xwalk_version: 1.0.0.1
  • Orientation: Landscape mode
  • Splash Screen: Turquoise background color
  • Splash Screen: Ready when the window.screen.show() method was called
  • Display: Opens the web application using the platform-specific convention for opening hyperlinks in the user agent (e.g., in a browser tab or a new window)

Expected results

  1. Install app successfully, no error, no warning message
  2. Display text "Manifest BMP" with turquoise background normally and the version is "1.0.0.1"
  3. Launch the app successfully and display "Splash Screen Landscape" with turquoise background and landscape screen
  4. The screen always stay the loading page

This sample demonstrates manifest members "icons", "xwalk_version", "orientation", "display", "xwalk_launch_screen"

  • Test values include: bmp format icons, four digits version, landscape orientation, broswer display, custom ready_when
  • Broswer: Opens the web application using the platform-specific convention for opening hyperlinks in the user agent (e.g., in a browser tab or a new window)
  • Xwalk_launch_screen members
    • Ready_when: The application readiness state at which to hide the launch screen
      • Custom:
        • The first visually non-empty paint has occurred
        • The window.screen.show() method was called
