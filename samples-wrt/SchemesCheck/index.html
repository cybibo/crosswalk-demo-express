This sample demonstrates enable scheme "app://", "mailto:", "market:", "sms:", "tel:", "geo:"

Pre-condition

  • Connect to network
  • Map Application is installed on device such as google map

Test Step

  1. Download app named like "schemescheck.apk" from https://github.com/crosswalk-project/demo-express/releases (The schemescheck demos apk might be included in .zip files in this page), then install and it
  2. Click "geo"
  3. Click "mailto"
  4. Click "open market"
  5. Click "send sms"
  6. Click "make call"

Expected Result

  1. Install apk and launch it successfully
  2. Popup the map app
  3. Popup the email app
  4. Popup the Market app
  5. Popup the Sms app
  6. Popup the Tel app
