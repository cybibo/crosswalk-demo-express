Crosswalk Manifest Demo 4

Download app named like "manifestdemo4.apk" from https://github.com/crosswalk-project/demo-express/releases (It might be included in .zip files in this page), then install it.
This demo shows following features can be configured by manifest.json:
  • Icon: A jpg format icon (purple background color)
  • Xwalk_version: 1.0.0.0.1
  • Orientation: Landscape mode
  • Splash Screen: Purple background color
  • Splash Screen: Landscape mode
  • Splash Screen: Ready when the first visually non-empty paint has occurred
  • Display: Minimal-ui provides the end-user with some means to access a minimal set of UI elements for controlling navigation (i.e., back, forward, reload, and perhaps some way of viewing the document's address)

Expected results

  1. Install app successfully, no error, no warning message
  2. Display text "Manifest JPG" with purple background normally and the version is "1.0.0.0.1"
  3. Display "Splash Screen Landscape" with purple background and landscape screen in Splash Screen
  4. Display some descriptions about manifest configuration with landscape screen
  5. Display status bar

This sample demonstrates manifest members "icons", "xwalk_version", "orientation", "display", "xwalk_launch_screen"

  • Test values include: jpg format icons, five digits version, landscape orientation, minimal-ui display, first-paint ready_when and landscape
  • Minimal-ui: This mode is similar to fullscreen, but provides the end-user with some means to access a minimal set of UI elements for controlling navigation (i.e., back, forward, reload, and perhaps some way of viewing the document's address)
  • Xwalk_launch_screen members
    • Ready_when: The application readiness state at which to hide the launch screen
      • First-paint:
        • The first visually non-empty paint has occurred
    • Landscape: The launch screen to use for landscape mode
